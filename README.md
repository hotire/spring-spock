## Block 

Spock에서는 given, when, then과 같은 코드를 block이라 한다. 

block은 test method에 최소한 하나는 있어야 한다. 

### given 

테스트에 필요한 환경을 설정하는 작업 

### when 

테스트 코드를 실행 

### then 

테스트 결과 검증 

### expect 

테스트 코드 실행 및 검증 (when + then)
